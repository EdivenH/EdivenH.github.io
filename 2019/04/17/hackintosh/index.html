记录一下自己黑苹果的折腾过程

发表于 2019-04-17 | 更新于 2019-04-23 | 分类于 Hackintosh | 阅读次数： | 阅读次数：

突然心血来潮，想折腾下macOS系统，奈何自己的口袋没钱买白苹果😭，只能弄个黑苹果玩玩，玩了快3天了，还是很不错的，没有什么大的bug，可以日常使用和开发，这里记录一下。

我的笔记本电脑是联想Y7000(i5-8300h 1050ti)

Note

固体硬盘是三星pm981型号的不建议安装到这个盘上，会出现卡死现象；

触摸板型号为ELAN的目前无法使用触摸板（估计以后也不会修复）

不会查看触摸板和硬盘型号的可以看最后有说明。

目前使用状况
  • 独立显卡无法驱动，也不用去尝试，目前无解
  • 有线网卡可以驱动，无线网卡无法驱动，需要使用的话可以更换无线网卡或者弄个USB无线网卡
  • hdml无输出，可以使用绿连的usba转hdml
安装

对于安装苹果其实不难，网上有很多集成EFI的镜像，我们要做的主要就是找到我们电脑适合的EFI文件，我的电脑是Y7000，我用的EFI文件，几乎完美，来自xiaoMGitHub，感谢作者🙏；我使用的镜像是黑果小兵的macOS 10.14.4，感谢黑果小兵🙏。

安装步骤

我们这里使用的是U盘来安装系统，U盘容量大于8G

  • BIOS设置（开机按F2）

    • 启用UEFI启动
    • 禁用安全启动
    • SATA模式设置为AHCI

  • 制作U盘启动盘

    • 下载etcher，选择我们下载的镜像，选择我们的U盘，Flash即可

  • 替换EFI文件

    • 我们把我们刚刚下载的EFI替换到U盘镜像中，这里我们用DG硬盘助手来操作（需要把EFI整个文件夹替换）

  • 安装macOS

    • 开机按F12，选择U盘启动
    • 没有问题的话进入到clover引导界面
    • 选择Boot macOS install from Install macOS Mojave

    • 正常会进入macOS使用工具页面
    • 选择磁盘工具，选择你要安装系统的硬盘，点击抹掉，名称随意，格式选择Mac OS 扩展(日志式)
    • 上一步执行完毕，选择安装系统，选择刚刚抹掉的盘，安装
    • 系统会重启，和第一步一样，选择U盘启动，进入clover引导界面
    • 这里注意选择Boot macOS Install from Mac，最后那个Mac是你安装系统的磁盘名

    • 等待安装，中间应该会有两次重启，依然和第一步一样选择U盘启动
    • 不出意外，大功告成
    • 喝杯咖啡，休息下☕️

安装过程就是这么简单，最重要的是找到适合自己的EFI，可以的话，自己配置也可以，哈哈

补充说明：

查看硬盘是否是PM981，打开设备管理器中查看

如上图就是PM981硬盘

查看触控板型号，打开设备管理器 - 人体输入设备 - I2C HID设备查看

如果为ELAN061B即不可驱动
